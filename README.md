How to publish
1. Create a GitHub repository named `maj3sty-hub.github.io`.
2. Clone it locally: `git clone https://github.com/maj3sty-hub/maj3sty-hub.github.io`
3. Copy these files and create the folders listed above.
4. Add your PDFs, images, slides, and screenshots.
5. Commit and push:
   - git add .
   - git commit -m "Initial Capstone portfolio for Francis Mahinda Munyiri"
   - git push origin main
6. Visit https://maj3sty-hub.github.io
